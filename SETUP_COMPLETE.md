## 🎯 How to Use Your Website

### Quick Start (Easiest):
```bash
# Just double-click this file:
start-all.bat
```

This will:
1. Install all dependencies
2. Start the backend on http://localhost:8000
3. Start the frontend on http://localhost:3000
4. Open both in separate terminal windows

### Manual Start:

**Frontend only:**
```bash
npm run dev
```

**Backend only:**
```bash
cd backend
uvicorn main:app --reload
```

## ⚠️ Before Full Use

1. **Setup .env file** with your API keys:
   ```env
   GEMINI_API_KEY=your_key
   AUTH_DATABASE_URL=your_db
   JWT_SECRET=your_secret
   ```

2. **Install Python dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Setup PostgreSQL database** (if using backend features)
